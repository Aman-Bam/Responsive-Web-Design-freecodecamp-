Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages.

Python's philosophy emphasizes code readability with its notable use of significant whitespace.

print("Hello, World!")
  • Easy to learn and use
  • Extensive standard library
Variables and Data Types

Python supports various data types including integers, floats, strings, and booleans.

Variables in Python are created when you assign a value to them.

x = 10 name = "Python"
  • Integers: whole numbers
  • Strings: text data
  • Lists: ordered collections
Control Structures

Python provides several control structures for decision making and iteration.

The most common are if statements, for loops, and while loops.

Indentation is crucial in Python as it defines code blocks.

if x > 0: print("Positive") for i in range(5): print(i)
Functions

Functions are reusable blocks of code that perform specific tasks.

They help organize code and avoid repetition.

  • Defined using the def keyword
  • Can accept parameters
Classes and Objects

Python is an object-oriented programming language.

Classes are blueprints for creating objects.

Objects are instances of classes with their own attributes and methods.
